  Live chat rooms over the Internet
 
The following links are to live text-based Internet chat rooms. Some of these sites are open at all times; others are have discussions at specific times.
All live chat rooms can be disappointing if you find yourself alone there -- so tell a friend to log in at a specific time and meet you there! Use the Feedback function if you'd like to add or modify a listing.

 

  AOL crohnscolitis chat

Bonnie hosts "crohnscolitis", which is a private room on AOL, and is open to any AOL subscriber. It runs on Sunday, Tuesday, and Wednesday evenings at 9pm EST. There is no set format or topic so feel free to jump in anywhere. The goal is to share experiences with others, and to seek and offer support. While the Specific Carbohydrate Diet is often discussed, Bonnie has success with the Protein Power by Doctors Eades. It a minimum of 60gms maximum 30gms carbohydrate.

This site has not been reviewed by the webmasters. Email us a review if you'd like!

+ Pros: TalkCity offers both free chat for "guests" as well as registered users. The site is very easy to use. It runs in Java, so you'll need to visit the site a few moments early to "update" your machine automagically with their system (under one minute at 28.8k). By the way, guest users can enter a fake email, thereby thwarting possible spam.

- Cons: A Java browser plug-in must be installed to talk with the group (easy).

 
  CDSI Crohn's Support Club

This club is for Crohn's disease patients and their family and friends. It is a great place for support, friendship and information from other Crohnies. This site uses text-based messaging "forum" style, and utilizes the Yahoo Messenger (Java applet) for live, one-on-one chat.

+ Pros: There always seems to be someone there.

- Cons: Requires member "registration" on Yahoo and a Java browser plug-in must be installed to talk one-on-one (easy).
